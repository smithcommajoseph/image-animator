ImageAnimator Examples

ImageAnimator elements are extremely easy to set up an require little besides the images themselves. Below are a few simple examples.

A list of all available params and their defaults are available in the README here

  • Non Looping
  • Looping
  • Accessing and Re-using an image animation

Non looping

If you have the following HTML

<div id="image-animator-1" class="image-animator-image-sequence">
	<img src="images/Flipbook0001.jpg" alt="some alt"/>
	<img src="images/Flipbook0002.jpg" alt="some alt"/>
	<img src="images/Flipbook0003.jpg" alt="some alt">
	...
</div>

And your Javascript looks like so

let imageAnimator = new ImageAnimator({
	el: document.getElementById('image-animator-1'),
	fps: 17
});

Looping

When creating a looping flipbook, you have a few more options, delay, loops.

Considering the following HTML

<div id="image-animator-2" class="image-animator-image-sequence">
	<img src="images/Flipbook0001.jpg" alt="some alt"/>
	<img src="images/Flipbook0002.jpg" alt="some alt"/>
	<img src="images/Flipbook0003.jpg" alt="some alt">
	...
</div>

And your Javascript looks like so

let imageAnimator = new ImageAnimator({
	el: document.getElementById('image-animator-1'),
	fps: 17,
	isLooping: true,
	//loops: 10,        // will run indefinitely if loops is not set
	//delay: 100        // sequences will run back to back, with no delay if not set
});

Accessing and Re-using a flipbook

You may re-set properties on a FlipBook object at anytime as long as you are storing a reference. and since you have access to the object's play, stop, and continue methods, you can start the ball rolling again.

let imageAnimator = new ImageAnimator({
	el: document.getElementById('image-animator-1'),
	fps: 17
});

// after done with playing...

imageAnimator.isLooping = true;
imageAnimator.play();
